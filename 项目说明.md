# Vue3 + Naive UI 项目说明

## 项目简介

这是一个基于 Vue 3 + Vite + Naive UI 的前端项目，包含九宫格首页和虚拟列表演示功能。

## 项目结构

```
vue3-naiveui/
├── src/
│   ├── views/              # 页面组件
│   │   ├── Home.vue       # 首页（九宫格布局）
│   │   └── VirtualList.vue # 虚拟列表演示页
│   ├── router/            # 路由配置
│   │   └── index.js       # 路由定义
│   ├── App.vue            # 根组件
│   ├── main.js            # 应用入口
│   └── style.css          # 全局样式
├── package.json           # 项目依赖
└── 项目说明.md           # 本文件
```

## 功能模块

### 1. 首页（九宫格布局）
- ✅ 虚拟列表 - 高性能大数据列表展示
- 🚧 数据表格 - 功能强大的数据表格组件
- 🚧 表单组件 - 丰富的表单输入组件
- 🚧 图表展示 - 数据可视化图表组件
- 🚧 文件上传 - 拖拽上传和文件管理
- 🚧 对话框 - 模态对话框和抽屉组件
- 🚧 消息通知 - 消息提示和通知组件
- 🚧 树形控件 - 树形结构数据展示
- 🚧 标签页 - 标签页和步骤条组件

### 2. 虚拟列表演示
- 📊 支持 10 万条数据流畅滚动
- 🔍 实时搜索过滤功能
- 🎨 美观的列表项设计
- 📈 数据统计显示

## 技术栈

- **框架**: Vue 3.5.39
- **构建工具**: Vite 5.4.0
- **UI 框架**: Naive UI 2.44.1
- **路由**: Vue Router 5.1.0

## 运行项目

```bash
# 安装依赖
npm install

# 运行开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 开发说明

### 添加新功能模块

1. 在 `src/views/` 下创建新的页面组件
2. 在 `src/router/index.js` 中添加路由配置
3. 在 `src/views/Home.vue` 中的 `modules` 数组中添加模块信息

### 路由配置

路由采用 memory 模式，配置文件位于 `src/router/index.js`。

## 注意事项

- 项目使用 Vue 3 Composition API
- 所有 Naive UI 组件都需要在 `n-config-provider` 和 `n-message-provider` 中使用
- 消息提示使用 `useMessage` hook
